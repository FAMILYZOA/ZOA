## :sunny: 서비스 소개

---

* 한줄로 확인하는 가족 소통앱!
* 매일 자신의 상태, 생각을 한줄로 남겨 가족들에게 안부를 전하세요! 오늘도 좋은 아침!
* 글 한줄로 남기기엔 부족함이 많다면 음성메시지는 어떤가요?
* 가족의 특별한 일정, 할 일을 등록하여 관리도 가능합니다!
